## 1. 序論：意味とは何か

「意味」はこれまで、言語哲学では命題の真理条件として、解釈学では文脈依存的な読みとして、認知科学ではスキーマや概念ネットワークとして論じられてきた。しかしこれらのアプローチは、意味が担う多層的な実在様式を構造的に記述する枠組みを持たず、言語以前、人間以外の生命が持つ意味についての説明が十分でなかった。

## 2. 基底空間：AEと意味の切り出し

### 2.1 不可知な基底（AE相）

八層構造モデルにおいて、**層1（A相：一般形而上層）** と **層8（E相：特殊形而上層）** は、認識限界の境界外──すなわち”意味の場”の外側──にある不可知の存在を配置する層である。

| 相 | 層 | 性質 | 意味論的含意 |
| :---: | :---: | :--- | :--- |
| **A** | 層1 | 時空的に普遍。宇宙の真の理・論理形式。認識不能。 | 意味の素材を供給するが、それ自体は意味として捉えられない。 |
| **E** | 層8 | 局所的な物理的実体。感性的経験では知り得ない物質基盤。 | Past.E は確定した実在として意味の生成を制約し続ける。 |

**命題 2.1（AEの意味論的地位）**：意味テンソル $M$ は直接 AE相を成分として含まない。しかし、意味はAEから切り出されておりAEはすべての意味成分の「制約的基底」として恒常的に作用している。A相E相（層1・8）の内容は $M_1$ および $M_8$ とは表記できず、もし、ある主体がこれらの内容に対する仮説や信念を保つ場合、それはAEの「写し」として BCD相（層2・3）にレンダリングされた成分（例：「神」「真理」「物自体」への信念）であって、AE自体が内包する成分を直接指すものではない。

### 2.2 クオリアによる切り出し

意味は AE という不可知の混沌から、**クオリア（層6）** というフィルターを介して切り出される。これは単なる比喩ではなく、以下の構造的必然から導かれる。

> 不可知な実在（AE）と認識可能な実在（BCD）の橋渡し役として、層6（クオリア）はその「客観的に不可知かつ主観的に可知」という二重性によって、唯一この媒介を遂行できる層として設計されている。

すなわち、意味 $M$ の生成は、

$$M = \mathrm{Cut}_{C}(\mathrm{AE}) \rightarrow \{M_2, M_3, M_4, M_5, M_6, M_7\}$$

という切り出し（Cut）操作として形式化できる。ここで $\mathrm{Cut}_C$ はクオリア（C相）による認識的切り出し操作を表す。

## 3. 意味テンソルの定義

### 3.1 基本形式

**定義 3.1（意味テンソル）**：対象 $x$（語・事物・出来事・関係・存在など）に対して、主体 $i$ における意味 $M_i(x)$ を次の六成分テンソルとして定義する。

$$\boxed{M_i(x) = (M_{i,2},\ M_{i,3},\ M_{i,4},\ M_{i,5},\ M_{i,6},\ M_{i,7})}$$

各成分は対応する層の実在様式において「対象 $x$ がどのような意味を持つか」を記述する。成分の値は実数的なスカラーではなく、各層固有の**意味命題の束**（あるいは命題的態度のベクトル）として理解される。

### 3.2 各成分の定義

| 成分 | 対応層 | 名称 | 実在の性質 | 意味論的内容 |
| :---: | :---: | :--- | :--- | :--- |
| $M_{i,2}$ | 層2 | **共有概念成分** | 形而上かつ形而下、客観、思弁 | 科学的定義・辞書的意味・宗教教義・社会通念。集団内で「正解」として検証・共有可能な意味。 |
| $M_{i,3}$ | 層3 | **個別信念成分** | 形而上かつ形而下、主観、思弁 | その対象に対する個人的な思い出・独自の解釈・人生哲学。他者には直接見えない「自分にとっての意味」。 |
| $M_{i,4}$ | 層4 | **社会規範成分** | 形而上かつ形而下、客観、矯正力あり | 法律上の定義・行政上の登録・組織内での役割。その意味が持つ「強制力」「実効性」「規範性」。 |
| $M_{i,5}$ | 層5 | **社会連関成分** | 形而上かつ形而下、客観、矯正力なし | 友人・家族間での暗黙の了解・情緒的な意味の共有・場の「空気」。柔軟なインターフェースとしての意味。 |
| $M_{i,6}$ | 層6 | **クオリア成分** | 客観的形而上・主観的形而上かつ形而下 | 経験される感覚の質そのもの。対象に付随する「質感・痛み・納得感・情動」。すべての意味の切り出しを行う駆動源。 |
| $M_{i,7}$ | 層7 | **純粋形而下成分** | 純粋形而下、客観、物理的実体 | 認識された物理的実体。言葉の「音」・書かれた「文字」・対象の「肉体」・物理的現象。意味を物理的に固定・伝播させる媒体。 |

### 3.3 具体例：「ペスという犬」の意味テンソル

意味テンソルの直観を具体的に示す。「ペス」という名の犬を飼っている主体 $i$ にとって：

$$M_i(\text{ペス}) = \left(\underbrace{[\text{犬・哺乳類・イヌ科…}]}_{M_{i,2}},\ \underbrace{[\text{子どもの頃一緒に遊んだ記憶…}]}_{M_{i,3}},\ \underbrace{[\text{役所に登録された飼い犬・予防接種済…}]}_{M_{i,4}},\ \underbrace{[\text{家族の一員…}]}_{M_{i,5}},\ \underbrace{[\text{温かい毛並み・声・匂い…}]}_{M_{i,6}},\ \underbrace{[\text{目の前にいるペスの肉体…}]}_{M_{i,7}}\right)$$

これら六つの成分が統合されて初めて「ペスという犬」という一つの意味 $M_i(\text{ペス})$ が成立する。ペスを知らない他者にとっては $M_{i,3}$・$M_{i,5}$・$M_{i,6}$ の成分が著しく異なり（ほぼゼロ）、意味テンソルは大きく乖離する。

## 4. 意味テンソルの構造的特性

### 4.1 主体依存性と座標性

**命題 4.1（意味の主体依存性）**：意味テンソル $M_i(x)$ は主体 $i$ の存在論的座標（物理的位置・過去経験 $Past.E_i$・クオリア構造・所属する社会層）に依存する。

$$M_i(x) \neq M_j(x) \quad \text{一般に（} i \neq j \text{ のとき）}$$

この主体依存性は欠陥ではなく、**構造的必然**である。$M_{i,6}$（クオリア成分）は原理的に他者に移転不能であり（主観的可知・客観的不可知）、$M_{i,3}$（個別信念成分）は各主体の過去経験の積分として形成される。したがって、完全に同一の意味テンソルを持つ二つの主体は存在しない。

**注**：$M_{i,2}$（共有概念成分）は「他者との収束を可能にする成分」として設計されているが、それでも主体 $i$ の認識経路に依存するため、$M_{i,2}$ と $L_{2,\text{consensus}}$（社会的合意としての層2）は一致するとは限らない（4.3参照）。

### 4.2 成分間の非独立性と相互作用

各成分は独立ではなく、相互に影響し合う。

| 相互作用 | 内容 |
| :--- | :--- |
| $M_{i,6} \rightarrow M_{i,3}$ | クオリア成分が個別信念を駆動する（実感から信念が生成される）。 |
| $M_{i,3} \rightarrow M_{i,2}$ | 個別信念が社会化・検証されることで共有概念へ昇華する（層3→層2の経路）。 |
| $M_{i,4} \rightarrow M_{i,6}$ | 社会規範成分が内面化されることで、クオリア層における「当たり前の感触」を形成する。 |
| $M_{i,7} \rightarrow M_{i,6}$ | 物理的接触・感覚器官を通じてクオリアが更新・書き換えられる。 |
| $M_{i,5} \leftrightarrow M_{i,6}$ | 情緒的連関（層5）とクオリア（層6）は双方向に共鳴・増幅し合う。 |

この相互作用の構造から、**意味テンソルは静的な配置ではなく、常に動的に更新され続ける生成体**であることが確認される。

### 4.3 $M_{i,2}$ の相対性：共有概念成分の「偽」問題

重要な構造的問題として、主体 $i$ が「世間一般の共有概念（層2）」として認識している $M_{i,2}$ が、実際の社会的合意 $L_{2,\text{consensus}}$ と乖離するケースがある。

$$\mathrm{Mismatch}[M_{i,2} \neq L_{2,\text{consensus}}]$$

これは「層2の主観的レンダリング」と呼ばれ、以下のような場合に発生する。

- **情報源の偏り**：閉鎖的コミュニティや特定の書物のみに接した結果、局所的な共有知が普遍的な共有知として内部でアンカリングされる（カルト宗教・陰謀論の構造）。
- **適用範囲の誤認**：家庭内や特定集団内での「常識」が、生命の総体における「常識」と同一視される（カレーにマヨネーズの例）。
- **歴史的時差**：かつての $L_{2,\text{consensus}}$ を現在のそれとして保持し続ける（時代遅れの「常識」）。

この問題は主体 $i$ の認識能力の限界から構造的に生じ得るものであり、道徳的欠陥ではない。しかしこれが価値の衝突・社会的対立の根底的要因となる点で、意味論的に極めて重要な現象である。

## 5. 時間軸との接続：意味テンソルの動態

八層構造モデルの動的実在論では、時間軸（Past / Present / Future）が実在の在り方を根本から規定する。意味テンソルもこの時間構造に服する。

### 5.1 $M_i^t(x)$：時刻 $t$ における意味テンソル

主体 $i$ の対象 $x$ に対する意味テンソルは時刻 $t$ の関数として表記される。

$$M_i^t(x) = \left(M_{i,2}^t,\ M_{i,3}^t,\ M_{i,4}^t,\ M_{i,5}^t,\ M_{i,6}^t,\ M_{i,7}^t\right)$$

### 5.2 過去・現在・未来における意味の構造

| 時制 | 意味テンソルの状態 | 構造的性質 |
| :---: | :--- | :--- |
| **Past.M** | $M_i^{t<0}(x)$：過去の意味テンソル | 過去の $E$（Past.E）によって基底は確定しているが、主体 $i$ の現在の認識からは不完全にしかアクセスできない（記憶・記録による不完全な媒介）。過去の意味は「改変不可能な制約」として現在の意味生成を規定する。 |
| **Present.M** | $M_i^{t=0}(x)$：現在の意味テンソル | クオリア（$M_{i,6}$）による現在進行形の切り出し。意味の「今ここ」。自由意志の作用点。 |
| **Future.M** | $M_i^{t>0}(x)$：未来の意味テンソル | 真に未決定。Present.M から出発する生成・思弁・行動（主に $M_{i,3}$ の構築と $M_{i,6}$ の駆動による）によって形成されていく。Future.M の不確定性こそが生命の創造的余地（自由）の存在論的根拠である。 |

**命題 5.1（意味の生成的非対称性）**：過去の意味テンソルは修正不能だが（Past.E の確定性）、未来の意味テンソルのみが真に未決定である。この非対称性が、意味の変化において「回顧的再解釈」（Past.M の $M_{i,3}$ 成分における再構成）と「未来生成的更新」（Future.M の能動的形成）を異なる行為として位置づける。

### 5.3 意味の発生史：C0から八層構造の成立まで

意味テンソルという概念は、生命の発生史と不可分に連動している。

| 段階 | 生命史 | 意味テンソルの状態 |
| :---: | :--- | :--- |
| 物質段階（AE） | RNA・タンパク質の自己複製。「意味」「主体」なし。 | $M$ は存在しない。純粋な AE の状態。 |
| C0（始点クオリア） | 「生きるべき」という始原の価値の最初の発火。 | $M_{i,6}$ の萌芽（原始的 $M_{i,6}^{C0}$）。他成分はゼロ。 |
| C1（受容体の獲得） | 単細胞生物。外界を「対象 D」として切り出す。 | $M_{i,6}$ と $M_{i,7}$ の最小構成が成立。$M = (0, 0, 0, 0, M_{i,6}, M_{i,7})$。 |

## 6. 意味テンソルと価値テンソルの関係

### 6.1 定義上の差異

意味テンソル $M$ と価値テンソル $\mathbf{V}$ は、クオリア（層6）によって同時に切り出されるが、その機能的役割は明確に区別される。

$$M_i(x) = (M_{i,2}, M_{i,3}, M_{i,4}, M_{i,5}, M_{i,6}, M_{i,7}) \quad \text{（対象 $x$ の構造的記述）}$$

$$\mathbf{V}_i(x) = (V_{i,a},\ V_{i,m},\ V_{i,\text{att}}) \quad \text{（対象 $x$ に対する生命の応答様式）}$$

| | 意味テンソル $M$ | 価値テンソル $\mathbf{V}$ |
| :--- | :--- | :--- |
| **問い** | 対象は「何であるか」 | 対象に「どう応答するか」 |
| **性質** | 構造的記述（存在論的） | 応答様式（動機論的） |
| **層の対応** | 層2〜7（BCD相）全体 | 親和極性・強度・注意（クオリア駆動） |
| **前後関係** | 先行する | $M$ の評価として後続する |

### 6.2 価値の衝突は意味のズレを前提とする

**命題 6.1（意味ズレと価値衝突の必然的連動）**：主体 $i$ と $j$ の間で価値の衝突が生じるとき、その多くの場合において、両者の意味テンソルに先行するズレ（$\Delta M$）が存在する。

$$\Delta M_{ij}(x) = M_i(x) - M_j(x)$$

$\Delta M_{ij} \neq \mathbf{0}$ のとき、両者は「同じ対象について語っているようでいて、実際には異なる対象（意味）について語っている」状態にある。

MLCS（多層接続構造解析）はこの衝突を以下のように分類する。

| 分類 | 条件 | 説明 |
| :--- | :--- | :--- |
| **Mismatch（層配置の誤認）** | 異なる層の成分を同一層として議論している | 例：$M_{i,7}$（物理的事実）を $M_{j,3}$（個人的信念）の問題として扱う偽の対立。 |
| **Antinomy（真の価値対立）** | $\Delta M$ が小さいにもかかわらず $V_a$ が逆方向 | 意味の土俵が揃っても、生への親和性が正反対になる状態。 |
| **Strength Antinomy（強度の対立）** | $V_a$ の方向は一致するが $V_{m}$ に大きな差分 | 「ちょっと避けたい」vs「死ぬほど避けたい」という価値強度の対立。政策優先順位の争いの本質。 |
| **Overreach（認識的越権）** | 仮想層（VL）で構築した他者モデルを真の $M_j$ と誤認 | 他者の意味を断言し、対話を打ち切る認識エラー。 |

## 7. 意味テンソルの動的生成：価値衝突による新規意味の創出

### 7.1 葛藤状態と意味の分節化

未分化なクオリア（$M_{i,6}$）から特定の「意味（$M_{i,2}, M_{i,3}$）」が生成される動力源は、価値テンソルの**親和極性 $V_a$ の葛藤**にある。

対象 $x$ に対して接近（$V_{a,\text{pos}}$）と回避（$V_{a,\text{neg}}$）が同時に発生する葛藤状態のとき、主体 $i$ はその緊張を解消するために、対象 $x$ をより細分化された概念として再定義（思弁的分節化）することを要請される。これにより、以前は単一の $M_{i,6}$ 成分として未分化であった意味が、$M_{i,3}$ 成分における新規概念の生成を経て、やがて $M_{i,2}$ 成分（共有概念）へと昇華される。

**意味の生成経路（標準的）**：
$$M_{i,6}(\text{切り出し}) \rightarrow M_{i,3}(\text{個別信念化}) \xrightarrow[\text{層5を介した共有}]{\text{層7への固定}} M_{i,2}(\text{共有概念化})$$

### 7.2 創造的誤解と意味の進化

主体 $i$ が私的言語（$M_{i,3}$・$M_{i,6}$の産物）を物理媒体（$M_{i,7}$）に固定し社会に開放するとき、受け手 $j$ はそれを自らの意味テンソルで再解釈する。

この「創造的誤解（Creative Misreading）」は欠陥ではなく、意味の多様性を保ちながら進化させる構造的エンジンである。完全に同一の意味テンソルが伝達されれば文化は停滞し、完全に別の意味テンソルとなれば対話が成立しない。その「ズレながら共鳴する」中間領域こそが、文化的生成のダイナミズムを生み出す。

## 8. 意味テンソルと文明・文化の接続

文化文明論で記述された「文明（層4）」「文化（層5）」「生命（層6）」の三層構造は、意味テンソルにおける成分の支配関係として形式的に記述できる。

| 社会的営み | 支配的成分 | 意味の性質 |
| :---: | :--- | :--- |
| **生命（根源）** | $M_{i,6}$（クオリア成分） | 移転不能。個固有の「実感」。全意味の熱源。 |
| **文化（本質）** | $M_{i,5}$（社会連関成分） | 暗黙知。身体化・情緒化された「場の意味」。言語化した瞬間に本質が失われる。 |
| **文明（外殻）** | $M_{i,2}$・$M_{i,4}$（共有概念・社会規範成分） | 形式知。論理化・明文化・再現可能な「透明な意味」。 |

**倫理的警鐘（文明論との接続）**：$M_{i,4}$（文明）の都合のために $M_{i,5}$（文化）が圧迫され、$M_{i,6}$（生命のクオリア）が均一化されるとき、意味テンソルは本来の熱源を失い「空洞化した構造」となる。これは構造的一元論における主従関係の逆転であり、**主観不可還元性公理（Ax. SI）** の侵犯に相当する。

## 9. 意味テンソルの不可知性の境界と倫理的含意

### 9.1 $M_1$・$M_8$ という「影成分」

前述のとおり、AE相（層1・8）は意味テンソルの直接成分ではない。しかし主体 $i$ は、不可知な層1に「神」「真理」「宇宙の理」を、層8に「物自体の確固たる実在」を投影し、それらを $M_{i,3}$（個別信念）または $M_{i,2}$（共有概念）として保持する。

これを「影成分」と呼ぶ：

$$M_{i,1}^{\text{shadow}} \subset M_{i,3} \cup M_{i,2} \quad,\quad M_{i,8}^{\text{shadow}} \subset M_{i,7} \cup M_{i,3}$$

これらの影成分は確かめようのないがゆえに、主体 $i$ にとって最も揺るぎない「実存的アンカー」として機能する。これが宗教・形而上学的信念の構造的説明となる。

### 9.2 私的言語と倫理的防壁

クオリア成分 $M_{i,6}$ は原理的に他者に移転不能である。この「通じなさ」は欠陥ではなく、主体 $i$ が他者のモデルに完全に吸収されることを防ぐ**倫理的防壁**である。

この防壁が、生成システム倫理学（GSE）における**主観不可還元性公理（Ax. SI）**の意味論的基盤をなす：$M_{i,6}$ は交換不可能であり、それを均一化しようとするいかなる社会的・文明的圧力も倫理的に禁止される。

## 10. 総括：意味テンソルの体系的位置づけ

意味テンソル $M_i(x) = (M_{i,2}, M_{i,3}, M_{i,4}, M_{i,5}, M_{i,6}, M_{i,7})$ は、八層構造モデルの体系において以下のように位置づけられる。

1. **存在論的基礎（構造生成実在論との接続）**：意味は AEという不可知の基底から、クオリア（C相）によって BCD相に切り出される多層的構造体である。意味の存在は BCD相の実在様式に完全に対応している。

2. **認識論的含意**：意味テンソルは主体依存的であり、完全な客観的共有は不可能である。しかし $M_{i,2}$ という共有概念成分が間主観的収束の可能性を担保し、文明・科学・言語を成立させる。

3. **倫理学的基礎（GSE・統合倫理哲学との接続）**：$M_{i,6}$（クオリア成分）の不可還元性が Ax. SI を基礎づけ、価値衝突の根底にある $\Delta M$ の可視化が MLCS解析を可能にし、意味の動的生成が「生成の継続こそが普遍善」という GSE 命題を支持する。

4. **文化文明論との接続**：文明（$M_{i,2}$・$M_{i,4}$）・文化（$M_{i,5}$）・生命（$M_{i,6}$）の階層は意味テンソルの成分支配構造として形式化される。文明が文化を、文化が生命のクオリアを圧殺するとき、意味テンソルは空洞化し、実在論的な「死」へと向かう。

5. **時間論との接続**：Past.M の制約性・Present.M の自由意志の作用点・Future.M の真の不確定性という三相構造が、意味の動的生成の時間的骨格をなす。
