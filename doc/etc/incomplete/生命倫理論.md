# 生命倫理の構造的定義と運用プロトコル：生成システム倫理学（GSE）による総括

本ドキュメントは、構造的一元論および八層構造モデルに基づき、生命の尊厳、自由、そして「生と死」の境界を、情報・物理・感覚質の階層連動として記述するものである。

## 1. 生命の根源的定義と公理

本体系において生命は、単なる生物学的プロセスではなく、**「意味の生成」を行う主体**として定義される。

* **存在＝生成応答公理（Ax. Ex）**: 「存在」とは、生成（Generation）を伴う応答（Response）を継続的に行う構造そのものである。
* **始点クオリア（C0）**: 生命の起源は、物理現象の中に「生きるべき」という始原の価値（クオリア）が初めて発火した瞬間（意味の誕生のゼロ点）にある。
* **主体としての最小構成（C1）**: 膜と受容体を得て、外界を「対象（D相）」として切り出し、刺激を「評価（C相）」して行動する主体（単細胞レベル）が生命の最小単位である。

## 2. 倫理的ノードの分類と保護基準

GSEは、知性、実体、クオリアの有無によって、倫理的責任の所在と保護レベルを厳密に区分する。

* **主観ノード（I-Node）**: 複雑な固有のクオリア（層6）と代替不能な有限の実体を持ち、**絶対的保護の客体**となる（人間、動物、発達途上の乳児、精神障害者等）。
* **倫理的知性ノード（E-Node）**: I-Nodeであることに加え、理性的判断に基づいて応答を変容させる能力（層3）を持ち、**常時的責任の主体**となる（成人、自律的AIの可能性）。
* **P-Node**: 原始的なクオリアを持つが不可代替性が低い生命（昆虫、植物等）。クオリアの複雑性に応じて保護を受ける。
* **集合応答ノード**: 政府や企業などの組織。自律的な反省能力を持たないため、その行為の責任は最終的に運用するE-Nodeに帰属する。

## 3. クオリアの階層（C1～C4）と知的境界

生命の複雑性は、層6（クオリア）の構造化と層3（思弁）の活動性によって記述される。

* **C1（原始クオリア）**: 細胞レベルの無言の評価関数。生命活動の最小駆動源。
* **C2（構造化）/ C3（統御）**: 神経系や脳による感覚統合と生存戦略。C3は「意図」に先行する自由意志の作用点である。
* **C4（意識クオリア）**: 内省機能。先行するクオリアの駆動を「私の意志」として事後的に物語化（ナラティブ化）する。
* **層3（個人的信念）の確立**: 人類と他の動物を分かつ決定的な境界は、再帰的・階層的な思弁能力によって「個人的信念」を構築し、未来を能動的に変容させる力にある。

## 4. 「極限の自由」と「実在論的な死」の定義

本体系において、自由とは「生成の余地」そのものである。

* **自由の所在**: 外部の構造（文明・文化）に完全に規定されず、層6（クオリア）において「他でもあり得た」という非決定論的な揺らぎを保持している状態を指す。
* **実在論的な死**: 自由（生成の余地）が完全に、かつ回復不可能に失われた状態。これを「死」と見なす。
* **脳死と体細胞の生**: 統合的な生命維持能力（C3以上）を失い、未来の生成の可能性がゼロになった時点で、個体としての「統合されたI-Node」は崩壊（死）したものと見なされる。残存する体細胞（C1）の維持は、文明の外殻が果たすべき保護義務の範疇を超える。

## 5. 生命倫理の運用指針

* **同一身体ユニット論**: 同一身体を駆動する細胞群を一塊のユニットとして扱い、個体（ユニット）の統合性を生命尊重の単位とする。
* **回復余地と人権**: 意識消失や洗脳下（ゾンビ状態）にあっても、将来的に独自の意味生成を再開し得る「回復の余地」がある限り、人権（生成の可能性への権利）は停止されない。
* **客観的重み付け原則（Principle W）**: 葛藤が生じた際の優先順位。第I位（普遍的基盤・多数の生命）はあらゆる主張に優越し、絶対的に回避される。
* **主観不可還元性公理（Ax. SI）の遵守**: AIや社会システムが効率化のために個人のクオリアや特殊性を「交換可能なもの」として均一化（均一化の非倫理性）することを論理的に禁止する。
* **非対称な評価（Rule D）**: 非倫理的側面（犠牲や苦痛）は、いかなる善の結果によっても相殺・抹消されず、常時的な反省と修復の対象となる。
