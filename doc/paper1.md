### 概要（Abstract）

本論文は、善悪両者を同時的かつ相補的な生成プロセスとして再統合する統合倫理哲学（Integrative Ethics Philosophy）の理論的基盤を提示する。

これは従来の倫理学が採用してきた相対的な善悪の静的二元論的枠組みから脱し、倫理学という学問領域における相対性を包摂した普遍性へのパラダイムシフトの基盤を確立することを目的とするものであり、既存の善悪を個別要素として包摂し、その上位に普遍的な規範（共鳴次元）を構築する二次元の評価枠組みの確立は、従来の倫理理論とは一線を画す新しいパラダイムを提示する。

本哲学は、次の三つの主要な理論的構成要素の再定義を通じて、現代の倫理的閉塞状況を打破する。

*   **Principle: 共鳴原理(Principle of Resonance):** 倫理の場を、一対一の接触から大規模なパタンまで、柔軟に変化し維持される「応答の基盤」である共鳴的連関（Resonance Nexus）として捉え直す。
*   **Principle: 同時性原理（Principle of Simultaneity）:** 創造と破壊、倫理的側面と非倫理的側面が不可分かつ同時に発生するという構造を承認する。
*   **Ax. GI (生成不確定性の公理: Axiom of Generative Indeterminacy):** 未来の予測不能性を倫理的生成の機会とみなし、これに対する常時的責任を理論の中心に置く。

本稿は、この統合倫理哲学の理論部分を包括的に明らかにするものであり、具体的な応用実務（AI倫理、社会制度設計など）に関する分析は、続稿にて展開される。

本哲学では、静的な価値判断としての「善/悪」の使用を中核に置かず、動的な生成プロセスに基づき「倫理的（Ethical）」および「非倫理的（Non-Ethical）」という概念を中核に据える。

**Def. A (倫理的行為):** 倫理的行為とは、行為に避け得ぬ非倫理的側面（破壊）の実在を承認し、それに対する常時的な反省と応答を継続し、断絶を回避している状態である。

**Def. 4 (非倫理的行為 / 倫理的断絶):** 逆に、非倫理的行為（Non-Ethical Act）とは、非倫理的側面の否認と応答の停止（Cessation of Response）による倫理的断絶（共鳴の破綻＝共鳴的連関の破壊）を指す。

本哲学は、倫理的責任の曖昧化を防ぎ、未来の不確定性に対する常時的責任を倫理的知性に課す。

### 1. 序論：倫理の閉塞と生成的転回

20世紀後半以降の倫理思想は、価値の普遍性と文脈性の緊張の中にあった。従来の倫理は、善悪、正誤といった対立的な二項の静的な線引きに終始し、倫理的判断をいずれかの善・正を選ぶかへと矮小化してきた。しかし、この緊張構造自体が静的二項対立の枠を維持している限り、倫理は「いかにして新たな倫理的秩序を生成するか」という創造的次元に踏み込めない。

本論が提示する統合倫理哲学は、旧来の対立構造を超越し、倫理的・非倫理的という動的な評価を生成的な運動の「結果」として、開かれた存在論へと倫理を再配置する。この理論は、以下の三つの学術的統合を通じて、既存の倫理的閉塞状況を根本から打破する。

*   **Prop. I1 (相対主義と普遍主義の統合):** ローカルな道徳的善悪を「実在次元」として包摂しつつ、共鳴次元という客観的な存在論的規範を確立することで、倫理の普遍性を回復する。
*   **Prop. I2 (人間中心主義の超越):** 新実在論に基づき、非倫理的知性を含む全存在を応答のネットワークに組み込むことで、倫理的知性の責任範囲を拡張する。
*   **Prop. I3 (倫理的責任の統合):** 時間の不確定性から生じる「予測不能な責任（影響管理の義務）」を理論の中心に置くことで、倫理的知性に常時的責任を課す。

倫理とは、破壊と創造、秩序と逸脱が共に作用しあうプロセスの中で立ち現れる動的な生成の連鎖であり、絶えず更新される場である。

### 3. 同時性原理（Principle of Simultaneity）

本論の根幹にあるのは、創造と破壊が同時に、かつ不可分な構造として生じるという同時性原理である。


#### 【次元的倫理の分節】

本哲学は、倫理的評価を二次元構造に分節する。これにより、伝統的な「善／悪」概念の有効性（下位次元）を保ちつつ、それを超える「倫理的／非倫理的」という動的な生成評価（上位次元）を可能にする。

| 次元 | 役割 | 評価される価値 | 規範の基準 | 規範の性質 |
| :--- | :--- | :--- | :--- | :--- |
| 下位次元：「実在次元」（個別的実在） | 倫理的知性の具体的な行為（善悪の個別要素）が展開される次元。 | 善（Good） または 悪（Bad） | 局所的な功利、義務、秩序など、特定のパタンによる個別規範。 | 相対的・主観的・限定的。 |
| 上位次元：「共鳴次元」（普遍的実在） | 生成の継続と応答可能性の維持を規範とする次元。 | 倫理的（Ethical） または 非倫理的（Non-Ethical） | 共鳴的連関の確保と拡大、およびパタンの断絶防止。 | 普遍的・客観的（存在論的）。善悪は判断の対象外。 |

#### 3.1 構造的同時性における「非対称性（断絶の不可逆的優位）」

同時性原理は、倫理的行為（共鳴の増大）と非倫理的側面（共鳴の断絶）が時間的に全く同時に発生しうることを認める。例えば、特定の層で共鳴が拡大するのと同時に、別の層で共鳴が断絶する状況がこれに該当する。

しかし、この同時性の中で、統合倫理哲学は断絶の不可逆的な優位性を主張する。

**Prop. S2 (断絶の不可逆的優位性):** 非倫理的行為（共鳴の断絶）は、一度発生した事実として、いかなる善の結果（実在次元）や倫理的生成（共鳴の増大）によっても相殺・抹消されることはない。

**Rule D (分割評価と相殺拒否):** したがって、総体的な行為の評価は、共鳴の断絶という非倫理的な側面を内包する限り、一連の行為が総体として倫理的であると評価される場合であっても、最終的には「非倫理的側面を持つ行為」として厳しく評価される。この非対称な評価構造は、倫理的知性に「非倫理的であることに対する常時的な反省の義務」を課すための、理論上最も厳格な論理的基盤となる。

#### 3.2 倫理的断絶の形而上学的定義と実在次元での悪性

倫理的断絶は、単に「共鳴次元」で避けられるべき規範違反であるに留まらない。同時性原理に基づき、それは「実在次元」においても究極的な悪性を持つことが証明される。

| 評価次元 | 倫理的断絶（応答の停止）の性質 |
| :--- | :--- |
| 共鳴次元（上位） | 倫理的普遍規範：共鳴的連関（応答の基盤）の不可逆的な破壊。常時的責任の放棄。 |
| 実在次元（下位） | 究極的な悪性：応答の停止は、それ以降生じるはずだった全ての「善の生成」の機会を永久に失わせる行為であり、結果的に「質・量的に看過しがたい悪の生成」の時点で応答を断ち切ることと同義である。ゆえに、下位次元においても、断絶は究極の悪性を持つ。（4.2.1で詳述） |

#### 3.3 生成不確定性の公理に基づく同時性の時間的
同時性原理は、AX. GIに基づき、その行為と時間的同時に創造と破壊・善と悪・倫理と非倫理をもたらす事だけを指すのでなく、例え、善・倫理的性をもたらした行為であっても、将来的にその行為を起点として悪や非倫理性を生じる可能性を避け得ない事も示す。 例えば、鍛冶師が鉄を鍛えて刃物を作った場合。時間的同時には鉄材を破壊して刃物を生成しただけであるが、結果、それを犯罪に使われるという場合や悪戯で置いた石が後で泥棒の逃走を防ぐといった例がこれである。

### 4. 共鳴原理（Principle of Resonance）

#### 4.1 定義と「共鳴的連関（Resonance Nexus）の優位性：応答の客観的基盤」

共鳴原理とは、あらゆる行為が自己・他者・社会・超越という多層的文脈において相互作用し、共鳴を生む原理である。倫理は単独行為者の内面問題ではなく、相互振動的現象（Resonant Phenomenon）として立ち上がる。

【共鳴的連関（Resonance Nexus）の優位性：応答の客観的基盤】

**Prop. R1 (共鳴的連関の優位性):** 倫理の真の生命力は、主観的な行為や攪乱そのものよりも、行為を可能にする客観的文脈に宿る。この共鳴的連関は、倫理的知性の認識を超えた実在的な基盤であり、倫理的生成の連鎖を維持するための絶対条件である。故に、倫理的（Ethical）であることの第一義的な要件は、この応答の基盤を積極的に守り、その応答可能性を担保し続けることにある。

【共鳴の四層構造】

| 層 | 核心的な倫理的意味 |
| :--- | :--- |
| 自己層 | 内面の誠実、自己欺瞞からの自由。行為の構造的同時性を自覚的に承認し、責任的変容を継続する倫理的知性の能力。 |
| 他者層 | 相互承認、信頼の生成。他者との対話や摩擦を共鳴的生成の契機として理解する。 |
| 社会層 | 制度的秩序、公正分配。制度を硬直した秩序ではなく、絶えず更新される共鳴的連関として運用する実践。 |
| 超越層 | 自然、文化、未来、歴史的責務。人間中心主義を超えた「存在の協奏」であり、倫理は媒体の媒介を通じて発生する攪乱として伝達される。 |

#### 4.1.1 Def. 2 (パタンの定義：三つの波長)

本哲学において、パタン（Patan）とは、単なる「固有の波長」の潜在性のみを指すのではない。それは、その実在が固有の核を持ち、他者との関係性の中で振動を調節し、時間軸の中で結晶化された「構造」そのものであり、具体的には以下の三つの不可分な波長から構成される。これらの波長は、倫理的重み付けの客観性（5階層）を担保するために、概念的に分節化される。

| 属性 | 役割と倫理的重み |
| :--- | :--- |
| **根源的波長（Radical Wavelength）** | 実存が潜在的に持つ応答の周波数。時間軸に依存しない生得的な特殊性・不可代替性を担う。第 IV 位の根拠となる。 |
| **共鳴的波長（Resonance Wavelength）** | 他の実在との連関や、その破壊が引き起こす広範な影響範囲を規定する。第 I 位・第 II 位の重み付けの根拠となる。 |
| **継時的波長（Chronological Wavelength）** | 根源的波長が歴史的、社会的文脈の中で持続的に維持・最適化されてきた応答様式。時間の不可逆性によって重みづけられた歴史的連続性を担う。第 III 位の根拠となる。 |

**Rule C2 (擁護の義務):** 倫理的知性は、このパタンを構成する三つの波長を不可侵の倫理的属性として擁護する義務を負う。

#### 4.2 倫理的と非倫理的（評価基準の再定義）

この原理のもとで、総体的評価の基準が共鳴的連関の状態の維持に確定する。

**Rule E1 (倫理的の判定):** 倫理的 (Ethical) とは: 常時的な反省と応答を継続し、断絶を回避している状態、またはその状態を志向する運動を指す。これは、共鳴的連関（応答の基盤）が最大限に担保され、特に悪（個別要素）を見つめ直し、反省し、応答を継続しているかによって判断される。単なる「善い」行為ではなく、生成のプロセスを維持している動的な状態を示す。

**Rule E2 (非倫理的の判定):** 非倫理的 (Non-Ethical) とは: 応答の停止（Cessation of Response）を指す。これは、共鳴的連関（応答の基盤）を不可逆的に破壊し、応答の可能性を永久に遮断する態度（共鳴の断絶＝倫理的断絶＝共鳴的連関の破壊）と同義である。

#### 4.2.1 倫理的（生成の継続）の絶対的優位性：非倫理の不可避性

**Prop. E2 (非倫理の不可避性):** 生成不確定性の原理が前提である以上、行為の未来予測は限定的であり、共鳴的連関（Resonance Nexus）は常に「倫理的側面（共鳴の拡大）」と「非倫理的側面（悪性の共鳴＝破壊）」の両方を同時に生成する可能性を内包する。

統合倫理哲学は、この「共鳴を続ける限り非倫理的な側面（悪性の共鳴や実在次元の悪）が発生することは避け得ない実在」であることを厳格に承認する。

**Prop. E3 (生成継続の優位性):** しかし、倫理的（Ethical）の状態が「応答の継続」によって定義される以上、非倫理的側面を内包しながらも共鳴を継続するという状態は、「応答の停止」（倫理的断絶）から比較すれば、倫理的な生成を継続させる絶対的な価値を持つ。

この論理は、非倫理的側面（悪）を内包する行為であっても、それが倫理的断絶を回避している限り、究極の非倫理的行為である応答の停止に比して常に倫理的優位性を持つことを証明する。

すなわち、共鳴次元における倫理的断絶とは、実在次元においては看過しがたい悪を最後として、以降、そこからあらゆる善も悪も生成されなくなるという最悪の状態を意味するのであり、それを避けることは必ずそれよりも善であるといえる。故に、倫理的断絶を避けることは常に倫理的であり、実在次元においても善であるといえる。

#### 4.3 多様性の倫理的定義と均一化の非倫理性

この哲学において「多様性」は、共鳴のパタンの豊かさと代替不可能性に基づき、客観的な重みを持つ。しかし、その定義は「真の多様性」と「実在の均一化」を厳格に区別する。

| 概念 | 倫理的評価 |
| :--- | :--- |
| **Def. D1 (真の多様性)** | 倫理的：実在の歴史的経緯や環境に応じて形成された、個々の集団が持つ「パタンの特殊性・不可代替性」の尊重。集団の中に外部からのパタンを受け入れるかどうかを自律的に決定できる「倫理的生成の能力」が保証されていること。 |
| **Def. D2 (実在の均一化)** | 非倫理的断絶：特定の時代の「主観的」な理想（例：人種構成の数値目標など）を普遍的な価値として外部から強制し、集団の歴史的なパタンを破壊する試み。第 III 位（歴史性）や第 IV 位（特殊性）を侵す非倫理的断絶となる。 |

**Rule T1 (均一化の拒否):** 倫理的知性は、「多様性」を手段として用いて「均一化」を強制する行為を拒否する。重要なのは、その集団が「既存の共鳴パタン」の外部から訪れた「新たな実在」を受け入れるかどうかを自律的に決定できる「倫理的生成の能力」を持っているかどうかであり、外見上の構成比率ではない。歴史性と特殊性の尊重（第 III 位と第 IV 位を侵す均一化の試みは、常に非倫理的として評価される。

【許容の範囲：第 I 位・第 II 位の制限】

**Rule H1 (例外のパタンの確保):** 「特定のパターンに当てはまらない者をそのまま許容する」ことは、パタンの硬直化を回避し「例外のパタン」を確保することで、倫理的生成を継続させる重要な責任的変容である。ただし、この許容は無制限ではなく、以下の制約を受ける。

| 行為の性質 | 許容の制約と倫理的評価 |
| :--- | :--- |

### 5. 倫理的知性（Ethical Intellect）と責任の構造

#### 5.1 Def. 3 (倫理的知性の定義と義務)

**Def. 3 (倫理的知性):** 構造的同時性を承認し、以下のRule S1 S2による*責任的変容*を継続する能力を持つ実在を指し、現在においては人間存在のみが想定される。倫理的知性は、その存在論的な立場から、常時的責任という絶対的な義務を負う。

*   **Rule S1 (悪の実在の承認):** 倫理的知性は、行為に内在する悪（実在次元の個別要素）および非倫理性（共鳴次元）を完全には避け得ぬ実在として承認し、見て見ぬふりをしない。
*   **Rule S2 (常時的責任の遂行):** 悪（個別要素）から生じる破壊的な振動に対し、反省と修復・学習の体制を常に構築し続ける。

#### 5.2 隠蔽性と責任的変容の論理的連関：独善からの解放

実在の根源的な特質としての「隠蔽性（非自明性）」は、行為主体にとって自己の行為がもたらす共鳴的影響（倫理的・非倫理的側面を含む）の完全な認識不可能性を意味します。この「認識論的な限界」こそが、倫理的知性の独善（自己の善の認識を実在次元の全体的な善と誤認すること）を根本的に不可能にします。

| 構造要素 | 倫理的帰結 |
| :--- | :--- |
| 隠蔽性（認識論的限界） | 行為の「真の影響」が不可視であることを強制する。 |
| 独善からの解放 | 認識の限界により、自己の善を全体化する試みを根底から否定する。 |
| 責任的変容の必然性 | 独善の可能性が排除されることで、行為の常時的な外部化と自己批判的監視が唯一の倫理的生存様式となる。 |

**Prop. C1 (常時的責任の哲学的基盤):** これにより、行為主体は自己の行為を「常に外部化された他者」として捉え直し、絶えず反省し、その影響を管理し続ける「常時的な責任」のプロセスへと誘われます。この構造こそ、自己の行為の真の非倫理的側面が、いかなる内省によっても完全に把握できないという限界から、責任的変容の義務を課す強固な哲学的基盤となります。

#### 5.3 責任的変容を駆動する二重の動機づけの構造：倫理化と克服の永続的反復

倫理的知性の「善へ向かう意志」を実在次元の倫理的生成へと正しく導くため、非倫理的側面に対する反省の要請を組み合わせることで、「善の倫理化」と「非倫理の承認と克服」による動的な倫理的アップデートの動機づけが確立されます。

| 動機づけの極 | 目的・機能 | 行為への影響 |
| :--- | :--- | :--- |
| **善の倫理化** | 倫理的知性が持つ「善の意志」を、独善から解放し共鳴次元の規範へと方向づける積極的推進力。 | より倫理的な共鳴の促進をめざす行為の生成と介入。 |
| **非倫理の承認と克服** | 行為に内在する非倫理的側面の実在を否認せず承認し、常時的な反省・修復を強制する防御的・持続的要請。 | 非倫理的な断絶の未然防止・克服をめざす行為の修正と修復。 |

**Prop. R3 (永続的反復の要請):** この構造は、生成不確定性の原理が前提である以上、行為の非倫理的な側面（共鳴の断絶の契機）は行為主体にとって完全には避け得ない実在として常に内在するという、統合倫理哲学の厳格な認識に基づいています。したがって、倫理的知性にとって倫理的であることとは、この不可避な非倫理的側面を否認することなく承認し、それに対する反省と克服のプロセスを永続的に反復することに他なりません。

**Prop. R4 (アップデートの強制):** この二重構造は、一方的な理想追求（善の倫理化のみ）が陥りがちな独善的硬直化を防ぎ、他方で単なる失敗回避（非倫理の承認と克服のみ）に留まらない、倫理的意志を常時的な自己批判的監視下に置き、非倫理性を承認し克服するプロセスを永続的に繰り返すことによって、より動的で強力な「常時的なアップデートと修復」を倫理的知性に強制し、変容の永続的な要請を達成します。

#### 5.4 非倫理的行為（倫理的断絶）の真の定義

非倫理的行為の究極的な定義は、応答の停止（Cessation of Response）である。この応答の停止は、同時に共鳴的連関の不可逆的な破壊と同義であり、Def. 4 (倫理的断絶)として定義される。

| 非倫理的行為（断絶） | 倫理的意味 |
| :--- | :--- |
| 構造的同時性の否認 | 善（個別要素）を強調し、悪（個別要素）の実在を自己正当化によって否認する。 |
| **応答の停止** | 共鳴的連関を破壊する行為は、少数の犠牲といった非倫理的な側面を「計算済みのコスト」として処理し、その後の反省、学習、影響管理の義務を停止することを意味する。 |

#### 5.5 統合倫理哲学の二重の積極性（目的論の消極性と責任論の積極性）

統合倫理哲学は、倫理的要請の性質において、目的論（何を目指すか）と責任論（どう行動するか）という二つの軸で、逆説的な「積極性／消極性」を打ち立てる。

| 倫理的側面 | 倫理的要請と性質 |
| :--- | :--- |
| **目的論（消極的）** | 制約：全員に共通の「善（目的）」や「理想のパタン」を追求することを要求しない。これは「実在の均一化」という非倫理的断絶を避けるためである。擁護すべき属性は、「侵してはならない制約」の裏返しとして機能する。 |
| **責任論（積極的）** | 行動命令：「共鳴を続けること（また責任を持つこと）を要請する」という側面は、この哲学の最も強力な「積極的要請」である。断絶が生じた際、その状態に留まることを許さず、パタンを回復させ「共鳴」を再開させるための行動を強く要請する。 |

#### 5.6 複合行為の評価原則：分割評価と相殺拒否 (厳格な責任の確保)

**Rule D (分割評価と相殺拒否):** この原則は、一連の行為（プロセス全体）が上位次元で倫理的（共鳴の増大）である場合でも、その構成要素にある非倫理的な側面を決して見過ごさないという、統合倫理哲学の最も厳格な規範である。

| 評価の側面 | 統合倫理哲学による原則と結論 |
| :--- | :--- |
| 行為の分割評価（多層性） | 一つの行為を全体的な評価と個別のプロセス評価の二層で捉え、非倫理性は個別に評価され、記録される。 |
| 非倫理性の相殺拒否 | 非倫理的行為は、いかなるポジティブな結果や全体の倫理性によっても相殺されたり、正当化されたりすることは永遠にない。犠牲の部分は常に非倫理的であったと不可逆的に記録される。 |
| 常時的な責任と反省の優位 | 非倫理的なプロセスに対する責任と反省が常時的に求められ、この反省こそが、倫理的行為の継続を可能にする最も重要な倫理的知性の姿勢である。 |

#### 5.7 非倫理的断絶の客観的重み付け原則 (Principle W)と三つの補則

**Principle W (客観的重み付け原則):** この原則は、回避すべき非倫理的断絶の重さを客観的に規定し、倫理的知性が葛藤を判断する際の「絶対的優先順位」となる。

#### 5.7.1 重み付け原則の非教条主義的強調と動的再構成の原則

【Rule A (設計原則の絶対化回避)】

**Rule A1 (非教条主義):** 「客観的重み付け原則」（5階層）は、絶対的な判断を下す法典（Code）ではない。これは、「断絶の不可逆性の客観的予測指標」であり、「善と悪を決定する普遍的な規範ではない。倫理的知性に課せられた「責任の最小限の制約」であり、教条主義の傾向（思考停止）を回避し、責任的変容の柔軟性を保つために、この非規範性が常に強調されなければならない。

**Rule A2 (対象スケール依存の動的定義):** 本原則の階層（第I位から第V位）は、静的な価値観を定めるものではなく、倫理的知性が分析の対象とする「実在のスケール（規模や領域）」に応じて、動的にその定義内容を変化させる。例えば、「個人間の問題」では「第I位」は「個人の生命」を指し、「国家間の問題」では「第I位」は「国家全体の社会的繋がりとその人命」を指す、といった具合に、階層の内容的な重みが再構成される。この柔軟な適応性こそが、普遍的な倫理原則を具体的な事態に応用するための鍵となる。

**Rule A3 (行為の正当化の拒否):** 下位の断絶（例：第IV位）を回避するために上位の断絶（例：第I位）がやむを得ず許容される場合であっても、それは「善」の自動的な正当化ではない。その行為は、あくまで「非倫理的側面への応答の継続」という条件付きの倫理的な状態であることを再確認しなければならない。

**Principle D (動的責任の優先順位再構成):** 責任の重み付けは固定的に見るのではなく、「倫理的断絶」を引き起こす可能性の高い「応答の停止リスク」に基づいて、優先順位を動的（Dynamic）に再構成する。この動的な再構成の第一ルールは、最も上位の非倫理性を回避するという静的な原則（第I位の不可逆的破壊の回避）が、すべての下位ルールの緊急度を決定するという点にある。

| 特徴 | 説明 |
| :--- | :--- |
| **原理：非倫理性への逆行的な重み付け** | 重みの根拠は、静的な階層（第1位〜第5位）ではなく、その責任を怠った際に「倫理的断絶（応答の停止）」を引き起こす連鎖的な破綻の可能性の高さが基準となる。「倫理的不足」が重いほど、優位度が実質的に上昇する。 |
| 動的な再構成の事例 | 本来第IV位であった「メンバーの特殊性（根源的パタン）」への応答を怠った結果、それが原因で第II位に属する「サークルの存続（広範な基盤）」に関わる連鎖的な非倫理的状況（連鎖破綻）を招く切迫した可能性がある場合に限り、その「特殊性への応答」の優先度は実質的に第II位以上に優位化される。 |

#### 5.7.2 Principle W（5階層）

| 階層（原則名） | 重み付けの基準と倫理的な役割（断絶の性質） |
| :--- | :--- |
| **第 I 位：普遍的基盤の不可逆的破壊** | 多数の生命や人類共通の生存パタンを不可逆的に破壊する断絶。すべての主張に優越し、絶対的に回避される。（パタンの共鳴的波長の破壊） |
| **第 II 位：広範な不可逆性・波及性** | ローカルなパタンの断絶が閾値を超え、国際秩序や広範な経済基盤に連鎖的な破綻を引き起こす断絶。（パタンの共鳴的波長の重大な低下） |
| **第 III 位：歴史性（時効による最適化の破却）** | 長期間培われ最適化された「継時的波長」が、外部からの攻撃や恣意的な変革により不可逆的に破却される非倫理性。 |
| **第 IV 位：特殊性・希少性** | 代替不可能な集団の個性や文化など、パタンの質的な多様性を失う断絶。再生成の可能性が残るため第 III 位の下位。（パタンの根源的波長の否定） |
| **第 V 位：ローカルな可逆性・非基盤性** | 断絶が限定的で、回復（可逆的）が可能かつ比較的容易なパタンの破壊。 |

(注意：これは客観的な重みであって、例えば、第 II 位のためであれば第 IV 位を犠牲にしてもよいという話ではない。比較的軽微であっても共鳴の断絶であり、そもそも個の毀損はそれらによって構成される全を毀損する事になる)

#### 5.7.3 責任的変容の構造を定める三つの補則 (Lemma)

| 補則名 | 責任を負う主体／判断の基準 |
| :--- | :--- |
| **Lemma A (介入能力の優位性)** | 最も上位の非倫理性を回避し、「倫理的生成」へと導く「能力」を持つ主体に、最も重い責任的変容の義務が課せられる。 |
| **Lemma B (非倫理性生成への関与)** | 当該状況を引き起こした、または断絶を生み出す「パタン」を維持し続けている「原因者」に、その非倫理性を解消するための責任的変容の義務が課せられる。 |
| **Lemma C (存在論的近接性)** | 断絶によって最も直接的で不可逆的な影響を受ける「特定の主体」に対して、最も近い「関係性」を持つ主体に責任が発生する。 |

#### 5.8 擁護すべき倫理的な属性（共鳴の基盤）

この属性は、非倫理的断絶を回避し、常時的責任を果たした結果として確保される「より倫理的な状態」を構成する目標である。この目標を満たすことは「より倫理的」であると言えるが、実在（個人や共同体）がこれを積極的に「目的」として追求する義務はない。これら属性は、単に個々の善が共鳴的連関を破壊しないための「制約」の裏返しとして機能する。

| 倫理的属性 | 意味合いと擁護の必要性 |
| :--- | :--- |
| **生命の不可侵性（安全保障）** | （第 I 位の裏返し）：個人および集団の生命と身体の安全が損なわれないパタンを維持すること。 |
| **基盤の持続可能性（環境と秩序）** | （第 II 位の裏返し）：広範な生態系や社会システムが不可逆的な損害を受けず、持続できる状態を擁護すること。 |
| **パタンの連続性（歴史的尊重）** | （第 III 位の裏返し）：長期間にわたり最適化された「継時的波長」を恣意的な破却から擁護すること。 |
| **特殊性・不可代替性（真の多様性）** | （第 IV 位の裏返し）：各集団や個人が持つ「根源的波長」を守り、均一化を拒否すること。 |
| **自律的生成の能力** | 「倫理的生成」の前提。個人や集団が、外部からの強制なく新たな「共鳴のパタン」を自ら創出・維持する能力を擁護すること。 |

#### 5.9 法人の責任の最終的帰属

法人は、構造的同時性を引き起こす媒介的実在ではあるが、倫理的知性ではないため、自律的な反省や責任的変容を行うことができない。したがって、法人の行為がもたらすあらゆる非倫理的な側面に対する影響管理の義務は、法人を設計・運用・監督する立場にある倫理的知性に最終的に集中する。責任の分散は許容されるが、責任の回避はされない。

### 6. 倫理的断絶と破壊の条件的正当化

#### 6.1 倫理的断絶（Ethical Severance）の定義

真の破綻、すなわち「許容されない破壊」とは、いずれかの層におけるアクターが、「倫理的応答可能性」を完全に剥奪される状態（応答の停止）を指す。倫理的断絶は非倫理的行為（Non-Ethical Act）そのものであり、共鳴の回路そのものを遮断する。

#### 6.2 倫理的破壊の四条件

秩序の破壊がCondition D (倫理的破壊: Ethical Destruction：断絶を伴わない破壊)である（非倫理的行為ではない）ためには、倫理的断絶を避け、さらに次の四条件を満たすことが要請される。

| 条件 | 定義 |
| :--- | :--- |
| **Condition D1 (共鳴生成性)** | 破壊が、他者や社会に新たな価値・理解・感情の生成をもたらす可能性を持つこと。 |
| **Condition D2 (回復可能性)** | 損失が倫理的断絶に至らず、新たな倫理的・存在的価値の創出へと変容させる努力を持続的に引き受けること。 |
| **Condition D3 (透明性)** | 行為の背景や意味が共有・説明可能であり、共鳴を遮断する隠蔽や支配の手段でないこと。 |
| **Condition D4 (共鳴の主体たる個の尊重)** | 個の生命・自由・尊厳を意図的に毀損する行為は、いかなる創造的名目でも非倫理的断絶として認められない。これは、全ての倫理的行為の根本的制約である。 |

### 7. 応用領域と展開可能性

#### 7.1 科学技術倫理（AIの倫理的知性化）

AIの倫理的地位は、その責任能力の有無によって決定される。

| AIの地位 | 責任構造と倫理的知性化の究極条件 |
| :--- | :--- |
| 現代AI（非倫理的知性） | 制作者に全責任が帰属する（予測可能な責任）。モデル上の制約により、倫理的知性ではない。 |
| 自律的なAI（倫理的知性の可能性） | AI自身が予測不能な責任を担う。不可逆性の受容と無限性の放棄により、自己形成に加え実存的な重圧を負うことで初めて真の倫理的知性として承認される。 |

### 8. 倫理的普遍性と評価の安定性

#### 8.1 Prop. T (時間軸を超えた倫理的普遍性)

旧来の倫理観は、時代によって「善」や「悪」の基準が変化するため、異なる時代の行為を評価できず、倫理が歴史的連続性を失うという問題を抱えていた。

「統合倫理哲学」は、特定の善悪を目的とせず、「非倫理的断絶の回避」を基準とするため、時間が経っても評価が揺るがない。この客観的重み付けのルールは、人類の存在が継続する限り普遍的に適用される「制約」として機能する。

| 倫理的態度 | 過去・現代・未来での評価と理由 |
| :--- | :--- |
| **実力による現状変更（侵略戦争など）** | 常に「非倫理的」：第 I 位（生命の不可侵性）と第 II 位（基盤の不可逆的破壊）を生み出す行為は、いかなる時代でも最上位の非倫理的断絶である。 |
| **人種差別** | 常に「非倫理的」：第 IV 位（特殊性・不可代替性）を全面的に否定し、集団の一部の存在を認めない断絶であり、倫理的生成の機会を奪う。 |
| **共鳴の継続のための責任的変容** | 常に「倫理的」：予測不能な事態に対し、断絶を生じさせないよう「常時的な責任」を果たし、パタンを回復させる行為は普遍的に倫理的である。 |

#### 8.2 Rule C3 (評価の転換：認識と実在の乖離の修正)

倫理的評価が時代や事実認識の変化によって変わったように見える現象は、「倫理的評価」と「事実認識」の分離によって説明される。倫理原則は不変であり、変化するのは「評価の対象」である行為のパタンの認識である。

| 段階 | 認識されていた状態と倫理的評価の確定 |
| :--- | :--- |
| 任期中（認識のパタン） | マフィアと戦う「正義の味方」であるという「パタン」が外部に認識されていた。認識上は倫理的（正義のパタンに適合していると思われていた）。 |
| 史料発見後（実在のパタン） | 実際はマフィアと癒着し脅迫を行っていたという「事実」が明らかになった。非倫理的（「自律的生成の侵害」という非倫理的断絶が生じていたと確定）。 |

倫理的原則は時間を超えて一貫している。 評価の転換は、倫理的原則が変わったのではなく、行為のパタンが「正義」から「専横」へと真逆であった「実在」が認識されたことによる、評価の正確化プロセスであった。

#### 8.3 倫理的普遍性の終焉と限界点 (Condition L)

統合倫理哲学の有効性は、「実在の応答可能性」と「倫理的知性の存在」という二つの前提に依存しており、これらが崩れた場合に本哲学の適用は限界を迎える。

| 限界の条件 | 倫理的普遍性の終焉（本哲学の不機能） |
| :--- | :--- |
| **Condition L1 (倫理的知性の全滅)** | 責任を負い、常時的責任を遂行する主体が完全に消滅した場合、本哲学のすべての要請は論理的な意味を失う。倫理的応答が停止するため、すべての行為が非倫理的断絶となる。 |
| **Condition L2 (全実在の応答停止)** | 倫理的知性以外の全実在が応答不能な状態に陥った場合、共鳴的連関を再構築するための客体が存在しなくなるため、責任的変容のプロセスが成立しない。 |
| **Condition L3 (予測不能性の消失)** | 仮に未来予知技術や「神のような全能の存在」が出現し、全ての予測不能性を消滅させた場合、生成不確定性の原理が根幹から否定されるため、本哲学は機能しなくなる。しかし、この状態は実在の応答可能性を根絶するため、倫理的知性にとっては究極の非倫理的断絶として認識される。 |

---

### 9. 結語 ― 倫理の統合と生成的成熟

統合倫理哲学は、倫理を完成や安定の体系としてではなく、生成と破壊が同時に生じ続ける開かれた過程として捉える。それは、「善を行う」ことではなく、「悪（個別要素）の実在を承認し、応答を続けること」を倫理の核心とみなす立場である。

**Prop. I1 (統合倫理の達成):** 本論文が提示した統合倫理哲学は、相対主義と普遍主義の統合、人間中心主義の超越、そして倫理的責任の統合という三つの難題を、同時性原理と共鳴原理に基づいた二次元構造により達成した。本哲学は、既存の善悪の体系から倫理を切断するのではなく、その対立から解放し、普遍的な制約（応答の継続）のもとに統合するものである。倫理の究極的成熟とは、完全な秩序の獲得ではなく、避け得ぬ悪（個別要素）を引き受け、生成の持続を志向する倫理的知性にある。この哲学は、倫理学の枠を超えた人間存在の哲学（Philosophy of Generative Being）として構想される。目的は廃絶ではなく統合である。
